Tile slider is a lightweight responsive slider for wordpress.

== Description ==

With inbuilt drag and drop admin panel its very easy to customize this plugin.

 [Tile Slider Demo](http://www.unizoe.com/demos/tile-slider/ "Plugin Demo")
 [Tile Slider Website](http://www.unizoe.com/products/tile-slider/ "Plugin Website")

== Installation ==

Install instructions : -

1. Upload `tile-slider` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto Tile Slider to manage sliders
4. Use as shortcode [tile-slider id=<id>] or as function tile_slider(array('id'=><id>))
